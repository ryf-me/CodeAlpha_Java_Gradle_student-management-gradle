# 🎓 Student Management System (Java + Gradle)

A beginner-friendly Java console application built using **Gradle** to demonstrate build automation, dependency management, unit testing, and CI/CD integration.

---

## 🚀 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ❌ Delete Student
- 🧪 Unit Testing using JUnit
- ⚙️ Automated Build with Gradle
- 🔄 CI/CD Pipeline using GitHub Actions

---

## 🛠️ Tech Stack

- Java (JDK 17/21)
- Gradle
- JUnit 5
- GitHub Actions

## ⚙️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/student-management-gradle.git
cd student-management-gradle

2. Run the Application
.\gradlew.bat run

🧪 Run Tests
.\gradlew.bat test

🏗️ Build Project
.\gradlew.bat build

🔄 CI/CD Pipeline

This project uses GitHub Actions to:

Automatically build the project
Run all test cases
